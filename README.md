# 📋 Task Management App

A dynamic task management application built with React.js, featuring real-time UI updates, centralized state management, and persistent data storage.

## 🔗 Links

- **Live Demo:** [View App](https://abdomousa22.github.io/To-Do-List/)
- **GitHub:** [View Code](https://github.com/AbdoMousa22/To-Do-List)

---

## ✨ Features

- ✅ Add, edit, and delete tasks in real time
- 🏷️ Organize tasks by status (pending / completed)
- 💾 Data persists after page refresh using Local Storage
- ⚡ Smooth UI updates with React Hooks
- 🌐 No prop-drilling thanks to Context API

---

## 🧰 Tech Stack

| Category | Technology |
|----------|-----------|
| Frontend | React.js, JavaScript (ES6+) |
| State Management | Context API, React Hooks |
| Persistence | Browser Local Storage |
| Styling | CSS3 |
| Deployment | Vercel / Netlify |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/AbdoMousa22/To-Do-List.git

# Install dependencies
cd To-Do-List
npm install

# Run the app
npm start
```
